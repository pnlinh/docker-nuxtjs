# docker-nuxtjs

## Build Setup

### Install [Makefile](https://makefiletutorial.com/) on your OS (recommend)

### To install project
```shell
make install
```

### To start project
```shell
make start
```

### To stop project
```shell
make stop
```

### To execute container
```shell
make shell
```

### To execute container with root user
```shell
make root
```

### To view container status
```shell
make ps
```

### To view container logs
```shell
make logs
```
